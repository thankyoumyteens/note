# 基本结构

```latex
\documentclass{article}
\usepackage{fontspec}
\usepackage{graphicx}
\usepackage{type1cm}
\usepackage{geometry}
\geometry{a4paper,left=1cm,right=1cm,top=3cm,bottom=3cm}
\usepackage[bold-style=ISO]{unicode-math}
% 添加heading=true, 使用中文版式
\usepackage[heading=true]{ctex}
% 生成pdf书签
\usepackage{hyperref}
% 自定义多级标题格式的宏包
\usepackage{titlesec} 
\titleformat{\section}[block]{\Huge\bfseries}{\arabic{section}}{1em}{}[]
\titleformat{\subsection}[block]{\huge\bfseries}{\arabic{section}.\arabic{subsection}}{1em}{}[]
\titleformat{\paragraph}[block]{\LARGE\bfseries}{[\arabic{paragraph}]}{1em}{}[]

\begin{document}
	\begin{flushleft}
		% 字体大小: 大
		\LARGE
		\begin{math}
			A \cap B \mbox{或者} AB
		\end{math}
		\\
		\begin{math}
			A \cup B \mbox{或者} A  B
		\end{math}
	\end{flushleft}
\end{document}

```
