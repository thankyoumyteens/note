# 将两个命令的输出作为另一个命令的输入

输出command1和command2的内容：
```
cat <(command1) <(command2) > outputfile
```
比较两个类似命令的输出
```
diff <(command1) <(command2) > outputfile
```
一次运行两个命令
```
cat <(command1 & command2) > outputfile
```

* `<()`运算符为每个命令设置命名管道, 将该命令的输出传递到命名管道并在命令行上传递名称, 等效于`>()`
* `>`表示覆盖原文件内容, 文件的日期也会自动更新
* `>>`表示追加内容, 会另起一行, 文件的日期也会自动更新
