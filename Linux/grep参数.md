# grep参数

1. -n  :显示行号
2. -o  :只显示匹配的内容
3. -q  :静默模式，没有任何输出，得用$?来判断执行成功没有，即有没有过滤到想要的内容
4. -l  :如果匹配成功，则只将文件名打印出来，失败则不打印
5. -A  :如果匹配成功，则将匹配行及其后n行一起打印出来
6. -B  :如果匹配成功，则将匹配行及其前n行一起打印出来
7. -C  :如果匹配成功，则将匹配行及其前后n行一起打印出来
8. -c  :如果匹配成功，则将匹配到的行数打印出来
9. -E  :等于egrep，扩展
10. -i  :忽略大小写
11. -v  :取反，不匹配
12. -w  :匹配单词
