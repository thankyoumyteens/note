# 第 1 课：建立卡提希娅参考图包

## 1. 本课要解决什么问题

你现在要做的是 **指定角色 AI 视频**，不是随机角色视频。

随机角色可以靠提示词直接生成，但指定角色不行。指定角色的核心问题是：

> 模型必须知道“卡提希娅到底长什么样”。

所以第 1 课不应该先写角色清单，也不应该先生成图，而是先建立一套可靠的视觉参考资料。

正确顺序是：

```text
参考图包
→ 角色控制清单
→ 角色标准图
→ 镜头关键帧
→ 图生视频
→ 剪辑成片
```

这节课的最终产物是：

```text
Cartethyia_Reference_Pack
```

也就是一个整理好的 **卡提希娅参考图包**。

## 什么是参考图包

参考图包不是随便存几张好看的图，而是专门服务 AI 生成的视觉资料库。

它要解决 6 个问题：

```text
1. 她的脸长什么样
2. 她的发型是什么结构
3. 她的服装是什么结构
4. 她的武器和配饰是什么
5. 她在官方 PV 里的镜头气质是什么
6. 哪些图最适合作为生成参考
```

如果参考图包混乱，后面生成时就会出现：

```text
脸不像
发型漂
服装简化
武器错误
配饰消失
画风不统一
变成普通银发二次元少女
```

## 什么是“核心参考图”

不是所有图都同等重要。

你最后要从 20–30 张图里精选出 **8 张核心参考图**。这 8 张图会成为后面所有生成工作的主要依据。

8 张核心图分别负责：

| 编号 | 类型              | 作用                           |
| ---- | ----------------- | ------------------------------ |
| 1    | 官方正面 / 半身   | 定脸、气质、上半身服装         |
| 2    | 官方全身          | 定身材比例、整体服装结构       |
| 3    | 侧脸或 3/4 脸     | 定脸部立体感和侧面轮廓         |
| 4    | 背影或大姿态      | 定背部服装、头发长度、整体姿态 |
| 5    | 眼部 / 表情特写   | 定眼睛、表情、情绪             |
| 6    | 武器 / 道具       | 定武器形状，避免生成错武器     |
| 7    | PV / 动画高光镜头 | 定电影感、光影、镜头氛围       |
| 8    | 服装 / 配饰细节   | 定头饰、衣服、饰品、飘带等细节 |

这 8 张图比 100 张杂图更重要。
