# 克隆声音

### 1. 获取语音数据

1. 玩家自制的数据集（最省事，推荐）: 在 2026 年，开源社区已经有很多针对二次元游戏的专属音频数据集。
   - ModelScope（魔搭社区）：国内拉取最快的地方。搜索“鸣潮”或“Wuthering Waves Dataset”，通常能找到专门为 TTS/SVC 整理好的 .7z 压缩包。
   - Hugging Face：全球最大的 AI 模型/数据仓库。搜索关键词：Wuthering Waves Character Voice 或 WuWa voices。
   - GitHub 项目：比如 AI-Hobbyist/WutheringWaves_Datasets 这种项目，会直接给出 ModelScope 的下载链接，包含了中、日、英、韩四国语言的切片音频。
2. 游戏 Wiki 媒体库（最快、音质极好）: 《鸣潮》的各类 Wiki 网站为了展示角色资料，通常会提取并存放单条语音。
   - Wuthering Waves Wiki (Fandom)：进入特定角色的 Voicelines 页面。例如进入 Yinlin/Voicelines 页面，可以直接试听并右键另存为 .ogg 或 .wav。
   - Bilibili 资源区：搜索“鸣潮 全角色语音包下载”，很多 Up 主会分享从游戏包体中解包出来的纯人声。
3. 硬核玩家方案：解包提取（如果你想拿到最新版本的角色）: 如果你想要最新版本（比如刚公测的还没人做数据集的角色），可以自己动手：
   - 工具：使用 FModel。
   - 路径：定位到游戏安装目录的 Client/Content/Paks。
   - AES Key：你需要去搜一下《鸣潮》当前的 AES 解密 Key（GitHub 上通常有实时更新的列表）。
   - 提取内容：在 FModel 导出 WwiseAudio_Generated 文件夹下的 .bnk 或 .pck 文件。
   - 转换：使用 vgmstream 工具将导出的音频流转换为 .wav 格式。

### 2. 拿到声音后，在 CosyVoice 里怎么用？

1. 准备素材：从上面下载的语音包里，选一段该角色最清晰、情感最稳定的 5-10 秒音频（尽量不要带大喊大叫的战斗语音）。
2. 上传音频：在 CosyVoice 的 “零样本克隆” 模块中，点击上传该音频。
3. 填写参考文本：在下方的“参考文本”框里，手动输入这段录音里原本说的内容。注意：这一步非常关键，它能极大提高克隆的相似度。
4. 输入目标文本：在最上面的输入框写你想让角色说的话。
5. 生成：点击生成语音。

如果你想让克隆出的语气更自然，建议在下载时多找几段：

- 一段冷静说话的（用于日常对话）。
- 一段战斗喊招的（用于需要气势的场景）。
- 在 CosyVoice 切换音色，就能实现同一个人在不同情绪下的“语气库”。
