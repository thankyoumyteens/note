# 扫描整数字面量

```java
// 整数字面量
// 1到9开头的是10进制
// 0x开头的是16进制
// 0开头的是8进制
// 末尾加U表示无符号数
// 末尾加L表示long类型
// 末尾加UL表示unsigned long类型
TOKEN: {
    <INTEGER: ["1"-"9"] (["0"-"9"])* ("U")? ("L")?
            | "0" ["x", "X"] (["0"-"9", "a"-"f", "A"-"F"])+ ("U")? ("L")?
            | "0" (["0"-"7"])* ("U")? ("L")?
            >
}
```
