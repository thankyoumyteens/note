# JavaCC 的正则

JavaCC 支持正则表达式, 它被用来定义词法分析器的规则。
JavaCC 支持以下正则

| 种类             | 示例            |
| ---------------- | --------------- |
| 固定字符串       | "int"           |
| 连接             | "ABC" "XYZ"     |
| 字符组           | \["X","Y","Z"\] |
| 限定范围的字符组 | ["0"-"9"]       |
| 排除型字符组     | ~["X","Y","Z"]  |
| 任意一个字符     | ~[]             |
| 重复 0 次或多次  | ("A")\*         |
| 重复 1 次或多次  | ("A")+          |
| 重复 n 次到 m 次 | ("A"){1,3}      |
| 重复 n 次        | ("A"){3}        |
| 重复 0 次或 1 次 | ("A")?          |
| 或               | "ABC" \| "XYZ"  |
