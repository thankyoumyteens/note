# 整数

求4210的二进制表示:

写成2的n次方相加的形式
```
4210 = 4096 + 24 = 4096 + 16 + 8
```
对照表
```
4096 2048 1024 512 256 128 64 32 16 8 4 2 1
```
表中有的数字对应写1, 没有写0

| 表   | 结果 |
|------|------|
| 4096 | 1    |
| 2048 | 0    |
| 1024 | 0    |
| 512  | 0    |
| 256  | 0    |
| 128  | 0    |
| 64   | 0    |
| 32   | 0    |
| 16   | 1    |
| 8    | 1    |
| 4    | 0    |
| 2    | 0    |
| 1    | 0    |

从上到下写出结果
```
bin(4210) = 1000000011000
```

# 小数

求5/32(0.15625)的二进制表示:

写成2的-n次方相加的形式
```
5/32 = 4/32 + 1/32 = 1/8 + 1/32
0.15625 = 0.125 + 0.03125
```
对照表
```
1/32 1/16 1/8 1/4 1/2
0.03125 0.0625 0.0125 0.25 0.5
```
表中有的数字对应写1, 没有写0

| 分数表 | 小数表  | 结果 |
|--------|---------|------|
| 1/32   | 0.03125 | 1    |
| 1/16   | 0.0625  | 0    |
| 1/8    | 0.0125  | 1    |
| 1/4    | 0.25    | 0    |
| 1/2    | 0.5     | 0    |

从下到上写出结果
```
bin(0.15625) = 0.00101
```
