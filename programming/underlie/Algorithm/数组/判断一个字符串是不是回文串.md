# 回文字符串

所谓回文字符串, 是指从左到右读和从右到左读完全相同的字符串。比如"level"或者"noon"等等就是回文串。

# 判断一个字符串是不是回文串

从两端向中间逼近

```c++
bool isPalindrome(string s) {
    int left = 0, right = s.length - 1;
    while (left < right) {
        if (s[left] != s[right])
            return false;
        left++; right--;
    }
    return true;
}
```
