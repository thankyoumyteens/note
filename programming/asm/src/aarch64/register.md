# 寄存器

## 通用寄存器

有 31 个通用寄存器: r0 至 r30。注意, `rxx` 记号通常用于描述汇编指令行为，不会出现在汇编代码中。

汇编代码中实际使用的:

1. x0 到 x30: 表示寄存器的 0 到 64 位。例如，x3 表示整个 r3 寄存器
2. w0 到 w30: 表示寄存器的低 32 位。例如，w3 表示 r3 寄存器的低 32 位

## 零寄存器

寄存器 xzr 和 wzr 被称为零寄存器。所谓零寄存器，就是指读取该寄存器的值，永远为 0。无法向该寄存器写入数值。其中 xzr 为 64 位的零寄存器，wzr 为 32 位的零寄存器。

由于精简指令集的原因，部分指令无法直接使用常数作为操作数。但是 0 作为一个特殊的常数经常出现在各种程序逻辑中，那么零寄存器的出现就可以省去将常数 0 存储到寄存器中的步骤。

`mov w0, wzr` 和 `mov w0, #0` 的效果相同, 都是把 w0 寄存器中的值设置为 0。

## 栈指针寄存器

sp 寄存器代表栈顶的内存地址。
