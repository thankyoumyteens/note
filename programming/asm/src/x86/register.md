# 寄存器

## 通用寄存器

通用寄存器是那些你可以根据自己的意愿使用的寄存器, 修改他们的值通常不会对计算机的运行造成很大的影响。通用寄存器最多的用途是计算。

1. ax/eax/rax 累加(Accumulator)寄存器: 常用于乘、除法和函数返回值。ax 表示寄存器一次能存取 16 位的数据, eax 表示寄存器一次能存取 32 位的数据, rax 表示寄存器一次能存取 64 位的数据
2. bx/ebx/rbx 基址(Base)寄存器: 常以它为基址来访问内存
3. cx/ecx/rcx 计数器(Counter)寄存器: 常做字符串和循环操作中的计数器
4. dx/edx/rdx 数据(Data)寄存器: 常用于乘, 除法和 I/O 指针
5. si/esi/rsi 来源索引(Source Index)寄存器: 存储器指针、串指令中的源操作数指针
6. di/edi/rdi 目的索引(Destination Index)寄存器: 存储器指针、串指令中的目的操作数指针

## 指针寄存器

1. bp/ebp/rbp 基址指针(Base Point)寄存器: 由被调函数保存, 指向函数的栈底
2. sp/esp/rsp 堆栈指针(Stack Point)寄存器: 指向栈顶
