# 系统调用

在汇编语言中, 进行系统调用(system call)的步骤: 

1. 准备系统调用号: 将系统调用的编号放入 `eax` 寄存器中
2. 准备参数: 将系统调用所需的参数放入特定的寄存器或通过栈传递。在 x86 架构中, 前六个参数通常通过 `ebx`, `ecx`, `edx`, `esi`, `edi`, 和 `ebp` 寄存器传递
3. 执行中断指令: 使用 `int 0x80` 指令(在 32 位保护模式下)或 `syscall` 指令(在 64 位长模式下)来触发系统调用中断
4. 获取结果: 系统调用的结果通常会返回到 `eax` 寄存器中。如果调用成功, `eax` 将包含返回值；如果调用失败, `eax` 将包含错误码
5. 错误处理: 如果系统调用失败, 可以使用另一个系统调用 `errno` 来获取更详细的错误信息

## exit

退出程序, 并返回状态码

- 调用号: 1
- 参数:
  - %ebx: 状态码
