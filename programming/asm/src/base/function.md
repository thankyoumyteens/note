# 函数

函数由如下几个不同部分组成。

## 函数名

函数名是一个符号，代表该函数代码的起始地址。

## 函数参数

函数参数是显示给函数以进行处理的数据项。

## 局部变量

局部变量是函数在进行处理时使用的数据存储区，在函数返回时即被废弃。对于某个函数的局部变量，程序中的任何其他函数都无法访问。

## 静态变量

静态变量也是函数进行处理时用到的数据存储区，但使用后不会被废弃，每当函数代码被执行时都可以重复使用。程序的任何其他部分都无法访问此数据。

## 全局变量

全局变量是函数进行处理时用到的、在函数之外管理的数据存储区。

## 返回地址

返回地址不能直接在函数中使用。返回地址告诉函数在执行完毕后, 应该从哪里继续执行。

在汇编语言中，cal1 指令会为你处理返回地址，ret 指令则按照该地址返回到调用函数的地方。

## 返回值

返回值是传回数据到调用者的主要方法。
