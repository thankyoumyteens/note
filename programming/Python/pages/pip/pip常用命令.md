# pip版本

```
pip -V
```

# 查看已经安装的第三方库

```
pip list
```

# 查看有新版本的第三方库

```
pip list --outdated
```

# pip安装第三方库

```
pip install 库名
pip install 库名==版本号
```

# pip升级第三方库

```
pip install --upgrade 库名
```

# 查看已安装库的详细信息

```
pip show 库名
```

# 卸载第三方库

```
pip uninstall 库名
pip uninstall 库名==版本号
```

# 更新pip

```
python -m pip install --upgrade pip
```
