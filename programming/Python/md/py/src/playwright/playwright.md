# Playwright

Playwright 是微软主导的浏览器自动化框架，主要用于：

1. **端到端测试 E2E**
   - 登录
   - 搜索
   - 下单
   - 表单提交
   - 后台管理系统流程测试

2. **网页自动化**
   - 自动点击
   - 自动填写
   - 自动下载
   - 自动截图
   - 自动抓取页面数据

3. **辅助复杂网页操作**
   - 先录制操作
   - 再改成稳定脚本
   - 最后可以反复运行

它支持 Chromium、Firefox、WebKit，也就是 Chrome、Edge、Firefox、Safari 相关场景基本都能覆盖。Playwright 官方也提供 Codegen 录制生成测试、Trace Viewer 调试回放、VS Code 插件等工具。
