# 图片通道数

在 OpenCV 中，图像的通道数（Channels）指的是构成图像的颜色成分的数量，不同的通道组合形成了不同的色彩空间。

常见的通道数情况如下：

- 单通道图像(灰度图): 通道数为 1，每个像素仅用一个值表示亮度（0-255），0 代表黑色，255 代表白色，中间值为不同程度的灰色
- 三通道图像: 最常见的是 BGR 格式(OpenCV 默认的彩色图像格式)，通道顺序为蓝(Blue)、绿(Green)、红(Red)，每个通道值范围为 0-255。此外，还有 RGB 格式(通道顺序为红、绿、蓝)，但 OpenCV 读取图像时默认会转为 BGR
- 四通道图像: 在 BGR 基础上增加了 Alpha 通道(A)，用于表示像素的透明度(0 表示完全透明，255 表示完全不透明)，即 BGRA 格式

```py
import cv2

img = cv2.imread('a.jpg')

# 显示图片的蓝色通道
cv2.imshow('blue', img[:, :, 0])
# 显示图片的绿色通道
cv2.imshow('green', img[:, :, 1])
# 显示图片的红色通道
cv2.imshow('red', img[:, :, 2])

cv2.waitKey()
```
