# 裁剪图片

在 OpenCV 中，裁剪图片可以通过数组切片操作实现，因为 OpenCV 读取的图像本质上是 NumPy 数组，裁剪就是选取数组的特定区域。

图像的坐标系统中，(x, y) 对应数组的 (列, 行)，裁剪时需要指定：

- 起始行 y1 和结束行 y2（不包含）
- 起始列 x1 和结束列 x2（不包含）

裁剪语法: `cropped_img = img[y1:y2, x1:x2]`。

```py
import cv2

img = cv2.imread('a.jpg')

# 裁剪 纵向第20行到第200行、横向第10列到第100列的区域
# 裁剪后的矩形四个角的坐标分别是:
# 左上角：(10, 20)
# 右上角：(100, 20)
# 左下角：(10, 200)
# 右下角：(100, 200)
crop = img[20:200, 10: 100]

cv2.imshow('crop', crop)

cv2.waitKey()
```
