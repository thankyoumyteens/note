# 模板匹配

模板匹配（Template Matching）是 OpenCV 中一种简单高效的目标定位方法，通过在输入图像中滑动模板图像，计算两者的相似度，找到与模板最匹配的区域。核心适用于 “目标在输入图像中仅出现一次、无旋转 / 缩放 / 形变” 的场景（如验证码识别、零件定位）。

核心原理：滑动窗口与相似度计算

1. 滑动窗口：将模板图像（如 100×100）作为 “窗口”，在输入图像上从左到右、从上到下逐像素滑动
2. 相似度计算：对每个窗口位置，计算模板与输入图像对应区域的 “匹配度”（用不同的匹配算法实现）
3. 定位最优区域：找到匹配度最高的窗口位置，即为模板在输入图像中的对应目标。
