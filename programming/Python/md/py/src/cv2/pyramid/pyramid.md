# 图像金字塔

图像金字塔是一种图像的多尺度表示方法，它通过对原始图像进行一系列的处理，生成一系列分辨率逐渐降低的图像集合，这些图像按照分辨率从高到低的顺序排列，形成类似金字塔的结构。

类型

- 高斯金字塔：主要用于图像的多尺度分析和处理，如图像匹配、特征提取和图像压缩等
- 拉普拉斯金字塔：主要用于图像的重建，也可用于图像增强、边缘检测、图像融合等，能捕捉图像中的细节和边缘信息

应用

- 图像融合：将两个图像的金字塔表示进行组合，然后通过金字塔重建生成无缝融合的结果，常用于创建全景图像或将两个不同焦点图像合成为一个全焦点图像
- 特征提取和目标检测：允许在多个尺度上搜索感兴趣的特征或对象，提高检测的准确性和鲁棒性，如在人脸检测中，可确保不会因为人脸尺度变化而漏检
- 图像压缩：通过拉普拉斯金字塔，可仅存储图像的差值信息实现高效压缩，还可选择性地丢弃一些高频信息以实现有损压缩，同时保持较好的视觉质量
