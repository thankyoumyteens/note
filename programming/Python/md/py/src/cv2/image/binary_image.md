# 二值图像

二值图像（Binary Image）是一种特殊的图像类型，其每个像素的取值只有两种可能：**黑**和**白**（或表示为数字 `0` 和 `255`，也可能是 `0` 和 `1`）。由于仅需 1 位（bit）即可表示一个像素，因此二值图像的存储空间远小于灰度图或彩色图。

常见应用场景：

1. 图像分割：将目标物体（前景）与背景分离，例如文字识别中提取文字区域
2. 轮廓提取：简化图像信息，突出物体的边缘和形状
3. 文档处理：如扫描的黑白文档、发票、文字表格等，仅需保留黑白信息即可
4. 机器视觉：用于简单的目标检测（如判断物体有无、位置等）

通常通过阈值化处理将灰度图转换为二值图像，例如 OpenCV 中的 `cv2.threshold()` 函数：

```python
import cv2

# 读取灰度图
gray = cv2.imread("image.jpg", cv2.IMREAD_GRAYSCALE)

# 阈值化：像素值 > 127 设为 255（白），否则设为 0（黑）
ret, binary = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)

cv2.imshow("Binary Image", binary)
cv2.waitKey(0)
```
