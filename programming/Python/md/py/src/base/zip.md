# 序列解压与重组操作

`zip(*locations)` 是一种常用的序列解压与重组操作

```py
# 示例数据
locations = [
    [50, 100, 150],  # x坐标数组
    [80, 130, 180]  # y坐标数组
]

# * 是 Python 中的解包运算符，用于将元组/列表中的元素拆分为独立的部分
# *locations等价于把locations拆分为两个独立数组：[50,100,150] 和 [80,130,180]
# zip(arr1, arr2) 的作用是把两个数组中的元素 “配对”，形成新的元组：(50, 80), (100, 130), (150, 180)
# arr是一个可迭代的对象, 类似于: [(50, 80), (100, 130), (150, 180)]
arr = zip(*locations)

for tp in arr:
    print(tp)

# 输出:
# (50, 80)
# (100, 130)
# (150, 180)
```
