# 控制元素

| 方法              | 说明                 |
| ------------------- | ---------------------- |
| clear()             | 清除文本           |
| send_keys (value)   | 模拟按键输入     |
| click()             | 单击元素           |
| submit()            | 用于提交表单     |
| get_attribute(name) | 获取元素属性值  |
| is_displayed()      | 设置该元素是否用户可见 |
| size                | 返回元素的尺寸  |
| text                | 获取元素的文本  |

例子
```py
#5.设置链接内容
element=browser.find_element_by_link_text("新闻")
element.click()

element=browser.find_element_by_link_text(""下团组"时间")
element.click()
```
