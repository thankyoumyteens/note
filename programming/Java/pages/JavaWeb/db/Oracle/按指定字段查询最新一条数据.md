# 按指定字段查询最新一条数据

Oracle 12c 及更高版本

```sql
SELECT *
FROM your_table
WHERE your_field = 'your_value'
ORDER BY date_field DESC
FETCH FIRST 1 ROW ONLY
```

---

较早版本的 Oracle

```sql
SELECT *
FROM (
    SELECT *
    FROM your_table
    WHERE your_field = 'your_value'
    ORDER BY date_field DESC
)
WHERE ROWNUM <= 1
```
