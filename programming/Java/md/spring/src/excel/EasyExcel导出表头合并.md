# EasyExcel导出表头合并

```java
@ExcelProperty(value = {"不合并01"}, index = 1)
private String col1;

@ExcelProperty(value = {"合并", "A"}, index = 2)
private String col2;

@ExcelProperty(value = {"合并", "B"}, index = 3)
private String col3;

@ExcelProperty(value = {"合并", "C"}, index = 4)
private String col4;

@ExcelProperty(value = {"不合并02"}, index = 5)
private String col5;
```
