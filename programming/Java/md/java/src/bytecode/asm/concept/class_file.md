# class 文件

.class 文件是个“结构化的二进制文件”。你所有的 Java 代码，编译后就是一个个的 .class 文件。

一个 ClassFile 结构，里面有：

- 魔数、版本
- 常量池（常量、字符串、类名、方法描述符等）
- 类信息（类名、父类、接口）
- 字段信息（Field）
- 方法信息（Method）
- 属性信息（注解、调试信息、字节码指令、行号表等）

ASM 干的事就是：

- 读这个结构 -> 按你的需求修改 -> 再写回去。

## 字节码与方法体

在 class 文件里：

- 每个 方法 对应一个 Code 属性

Code 属性里有：

- `max_stack`：操作数栈最大深度
- `max_locals`：局部变量表大小
- `code[]`：一串字节码指令
- 异常表、行号表等
