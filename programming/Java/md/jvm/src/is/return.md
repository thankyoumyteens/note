# 方法返回指令

| 操作码  | 操作数 | 操作数栈-执行前 | 操作数栈-执行后 | 操作                   |
| ------- | ------ | --------------- | --------------- | ---------------------- |
| ireturn | -      | value           | -               | 从当前方法返回 int     |
| lreturn | -      | value           | -               | 从当前方法返回 long    |
| freturn | -      | value           | -               | 从当前方法返回 float   |
| dreturn | -      | value           | -               | 从当前方法返回 double  |
| areturn | -      | value           | -               | 从当前方法返回对象引用 |
| return  | -      | -               | -               | 从当前方法返回 void    |

说明:

- ireturn: 当前方法的返回值必须为 boolean、short、char 或者 int 类型，value 必须是一个 int 类型的数据。如果执行过程当中没有异常被抛出的话，那 value 将从当前栈帧中出栈，然后压入到调用者栈帧的操作数栈中，在当前栈帧操作数栈中所有其他的值都将会被丢弃掉。指令执行后，解释器会恢复调用者的栈帧，并且把程序控制权交回到调用者
