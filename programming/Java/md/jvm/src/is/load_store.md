# 加载和存储指令

- i 代表对 int 类型的数据操作
- l 代表 long
- s 代表 short
- b 代表 byte
- c 代表 char
- f 代表 float
- d 代表 double
- a 代表 reference

比如 iload 表示从局部变量表加载一个 int 类型值到操作数栈, fload 表示从局部变量表加载一个 float 类型值到操作数栈中。

## 将一个局部变量加载到操作栈

包含的指令: iload, `iload_<n>`, lload, `lload_<n>`, fload, `fload_<n>`, dload, `dload_<n>`, aload, `aload_<n>`。

| 操作码      | 操作数 | 操作数栈-执行前 | 操作数栈-执行后 | 操作                                         |
| ----------- | ------ | --------------- | --------------- | -------------------------------------------- |
| iload       | index  | -               | value           | 从局部变量表加载一个 int 类型值到操作数栈中  |
| `iload_<n>` | -      | -               | value           | 从局部变量表加载一个 long 类型值到操作数栈中 |

- iload 说明: index 是一个代表当前栈帧中局部变量表的索引的无符号 byte 类型整数，index 作为索引定位的局部变量必须为 int 类型，记为 value。指令执行后，value 将会压入到操作数栈栈顶。
- `iload_<n>`说明: `<n>`代表一个当前栈帧中局部变量表的索引值，`<n>`作为索引定位的局部变量必须为 int 类型，记作 value。指令执行后，value 将会压入到操作数栈栈顶
