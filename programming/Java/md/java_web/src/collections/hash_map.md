# HashMap

实现原理: 数组 + 链表。从 JDK 8 开始, 当链表长度大于 8 且数组长度大于 64 时, 链表转换为红黑树。

无参构造方法:

```java
public HashMap() {
    // 并不初始化数组, 默认为空
    // 初始化加载因子为0.75
    this.loadFactor = DEFAULT_LOAD_FACTOR;
}
```

put 方法的流程:

1. 判断数组是否为空
   - 为空, 调用 resize 方法, 初始化数组, 长度为 16
2. 根据哈希算法计算 key 在数组中的索引
3. 索引处的元素是否为空
   - 为空, 直接插入
   - 不为空, 判断 key 是否存在
     - 存在, 覆盖 value
     - 不存在, 添加到链表/红黑树中
4. 判断元素个数是否达到扩容的阈值
   - 是, 调用 resize 方法扩容

## 扩容

扩容的时机: 添加的元素个数超过 `容量 * 加载因子`。容量默认 16, 加载因子默认 0.75。
