# 数据结构

## 反转一个单链表

示例: 输入: 1->2->3->4->5->NULL 输出: 5->4->3->2->1->NULL

1. 初始化三个指针：
   - prev 初始化为 NULL
   - cur 初始化为链表的头节点
   - next 用于临时保存当前节点的下一个节点
2. 遍历链表，在每次迭代中：
   1. 保存 cur 的下一个节点到 next
   2. 改变 cur 的指针方向，使其指向前一个节点 prev
   3. 更新 prev 和 cur 的位置，prev 移动到 cur，cur 移动到 next
3. 当 cur 为 NULL 时，遍历结束，此时 prev 就是反转后链表的头节点

```java
public static Node reverse(Node node) {
    Node cur = node;
    Node next = null;
    Node prev = null;
    while (cur != null) {
        // 保存 cur 的下一个节点到 next
        next = cur.next;
        // 改变 cur 的指针方向，使其指向前一个节点 prev
        cur.next = prev;
        // prev 移动到 cur
        prev = cur;
        // cur 移动到 next
        cur = next;
    }
    // prev 就是反转后链表的头节点
    return prev;
}
```
