# 递归查询所有下级

数据:

| id   | name   | parent_id |
| ---- | ------ | --------- |
| 1111 | 北京市 | 0000      |
| 2222 | 昌平区 | 1111      |
| 3333 | 沙河镇 | 2222      |
| 4444 | 河北省 | 0000      |
| 5555 | 廊坊市 | 4444      |

查询:

```sql
select *
from area_info
start with id = 1111 connect
by prior id = parent_id
```

| ---- | ------ | --------- |
