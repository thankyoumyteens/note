# 用户

用户的组成: 用户名和白名单, 格式 `用户名@'白名单'`。

```sh
# 所有ip都可以远程连接test用户
test@'%'
# 只有192.168.1.100这个ip可以远程连接test用户
test@'192.168.1.100'
# 从192.168.1.1到192.168.1.255的ip可以远程连接test用户
test@'192.168.1.%'
# 从192.168.1.50到192.168.1.59的ip可以远程连接test用户
test@'192.168.1.5%'
# 只能本地登陆
test@'localhost'
# 只能本地登陆
test@'127.0.0.1'
```
