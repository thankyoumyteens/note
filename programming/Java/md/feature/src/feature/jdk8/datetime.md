# Date-Time API

Java 8 引入的 Date-Time API（位于 java.time 包下）是对传统日期时间 API（如 java.util.Date、Calendar）的彻底重构，解决了旧 API 线程不安全、设计混乱（月份从 0 开始）、功能有限等问题。新 API 基于 不可变对象 和 fluent 风格 ，提供了清晰的类职责划分和丰富的日期时间处理能力。

| 类名                | 描述                                                       | 示例                                           |
| ------------------- | ---------------------------------------------------------- | ---------------------------------------------- |
| `LocalDate`         | 只包含日期（年、月、日），无时间和时区                     | `2023-10-05`                                   |
| `LocalTime`         | 只包含时间（时、分、秒、纳秒），无日期和时区               | `14:30:25.123456789`                           |
| `LocalDateTime`     | 包含日期和时间，无时区                                     | `2023-10-05T14:30:25`                          |
| `ZonedDateTime`     | 包含日期、时间和时区（带时区的完整时间）                   | `2023-10-05T14:30:25+08:00[Asia/Shanghai]`     |
| `Instant`           | 时间戳（UTC 时间，从 1970-01-01 00:00:00 开始的秒/纳秒数） | `2023-10-05T06:30:25Z`（UTC 时间）             |
| `Duration`          | 表示两个时间点之间的时间间隔（秒、纳秒）                   | `2小时30分钟`（基于 `LocalTime` 或 `Instant`） |
| `Period`            | 表示两个日期之间的日期间隔（年、月、日）                   | `2年3个月5天`（基于 `LocalDate`）              |
| `DateTimeFormatter` | 日期时间格式化/解析工具（线程安全）                        | `DateTimeFormatter.ofPattern("yyyy-MM-dd")`    |
