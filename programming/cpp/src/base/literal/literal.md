# 字面量

字面量用于在代码中编写数字或字符串。

C++支持的标准字面量

| 类型                   | 示例      | 说明 |
| ---------------------- | --------- | ---- |
| 十进制字面量           | 123       | -    |
| 八进制字面量           | 0173      | -    |
| 十六进制字面量         | 0x7B      | -    |
| 二进制字面量           | 0b1111011 | -    |
| 浮点数                 | 3.14f     | -    |
| 双精度浮点数           | 3.14      | -    |
| 字符                   | 'a'       | -    |
| 以 `\0` 结尾的字符数组 | "abcde"   | -    |

可以在数值字面量中使用数字分隔符, 数字分隔符是一个单引号。例如: 23456789 可以写成 23'456'789。
