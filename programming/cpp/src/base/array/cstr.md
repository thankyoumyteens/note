# C 风格字符串

C 语言约定: 如果字符型(char)数组的未尾包含了空字符 `\0`(也就是 0), 那么该数组就是一个字符串。

```cpp
char *str = "hello";

// 获取字符串的长度
std::cout << strlen(str);
```

注意:

- 字符串的结尾标志是 0，按照约定，在处理字符串的时候，会从起始位置开始搜索 0，一直找下去，找到为止(不会判断数组是否越界)
- 结尾标志 0 后面的都是垃圾内容
- 字符串在每次使用前都要初化
- 不要在子函数中对字符指针用 sizeof 运算，所以，不能在子函数中对传入的字符串进行初始化，除非字符串的长度也作为参数传入到了子函数中
