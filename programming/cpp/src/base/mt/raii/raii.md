# 锁管理工具

为了更安全地使用互斥锁（避免忘记解锁导致死锁），C++11 提供了 RAII 风格的锁管理工具：

- std::lock_guard: 构造时自动锁定，析构时自动解锁（离开作用域时）
  - 特点：简单、高效，适合简单的临界区保护
  - 局限性：锁定后不能手动解锁，生命周期固定
- std::unique_lock: 更灵活的锁管理，支持手动锁定 / 解锁、延迟锁定等
  - 特点：可以在需要时锁定，不需要时提前解锁
  - 适用场景：需要控制锁的生命周期、条件变量等复杂场景

RAII（Resource Acquisition Is Initialization，资源获取即初始化）是 C++ 中一种管理资源的编程范式，其核心思想是将资源的生命周期与对象的生命周期绑定：在对象创建时获取资源，在对象析构时自动释放资源，从而确保资源的正确管理。
