# 指针运算

如果声明一个指向 int 的指针，然后将这个指针递增 1，那么这个指针在内存中向前移动 1 个 int 的大小，而不是 1 个字节。

```cpp
int *arr = new int[10];

// 等价于 arr[2] = 100;
*(arr + 2) = 100;
```

指针运算的另一个有用应用是减法运算。将一个指针减去另一个同类型的指针，得到的是两个指针之间指针指向的类型的元素个数，而不是两个指针之间字节数的绝对值。

```cpp
int *arr = new int[10];

int *end = &arr[9];

// length = 9
long length = end - arr;
```
