# 函数

在 C++中, 为让其他代码使用某个函数, 首先应该声明该函数。如果函数在某个特定的文件内部使用, 通常会在源文件中声明并定义这个函数。如果函数是供其他模块或文件使用的, 通常会在头文件中声明函数, 并在源文件中定义函数。

函数声明通常称为 函数原型 或 函数头, 以强调这代表函数的访问方式, 而不是具体代码。

函数签名 是指将函数名与形参列表组合在一起, 但不包括返回类型。

## 当前函数的名称

每个函数都有一个预定义的局部变量 `__func__`, 其中包含当前函数的名称。

```cpp
#include <iostream>

int main() {
    // 输出: main
    std::cout << __func__ << std::endl;
    return 0;
}
```
