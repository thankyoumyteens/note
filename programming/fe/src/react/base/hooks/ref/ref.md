# useRef

`useRef` 可以帮助你在函数组件中处理各种需要持久化存储和 DOM 操作的场景。通过 `useRef`, 你可以避免不必要的重新渲染, 并且能够方便地访问和操作 DOM 元素。

主要用途：

1. 持久存储: `useRef` 可以用来存储任何可以在整个组件生命周期内持久化的数据, ref 的值改变不会触发组件的重新渲染
2. DOM 访问: 可以使用 `useRef` 来获取对 DOM 元素的直接引用, 类似于类组件中的 `ref` 属性
3. 跨渲染保持数据: `useRef` 可以用来保存上一次渲染的数据

`useRef` 返回一个对象, 该对象有一个 `.current` 属性。你可以在 `.current` 上存储任何值, 包括原始值、对象、函数等。
