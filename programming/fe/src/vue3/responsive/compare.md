# ref 和 reactive 的区别

1. **数据类型**：

   - `ref` 可以接收基本类型值(如数字、字符串等)，也可以接收对象
   - `reactive` 只能接收对象(包括数组和函数等)作为参数

2. **使用方式**：

   - 使用 `ref` 创建的数据需要通过 `.value` 来访问或设置
   - `reactive` 返回的对象可以直接访问和修改，无需额外的 `.value`

3. **失去响应性**：

   - 当从 `reactive` 对象中解构出属性时，这些属性将失去响应性。可以使用 `toRefs` 来保持解构后的属性为响应式。给 `reactive` 对象赋值为新的对象也会失去响应性。可以使用 `Object.assign` 来把新对象的所有字段复制给 `reactive` 对象
   - `ref` 在解构后仍然保持响应性，因为每个 `ref` 实例都是独立的响应式单元
