# GPT、Claude、Llama 等主流模型差异

| 模型家族 | 最核心定位                                                              |
| -------- | ----------------------------------------------------------------------- |
| GPT      | 商业闭源最强通用模型之一，适合复杂 Agent、代码、工具调用、生产应用      |
| Claude   | 商业闭源强推理/长上下文/写作/代码模型，适合文档、代码协作、复杂业务分析 |
| Llama    | 开权重模型，适合私有化部署、低成本规模化、自定义微调、企业内网场景      |
| Gemini   | Google 模型，强多模态、长上下文、Google 生态集成                        |
| Qwen     | 阿里模型，中文、代码、Agent、开源/商业并行，国内生态友好                |
| Mistral  | 欧洲模型厂商，强调开放模型、企业部署、低延迟和可控性                    |
| DeepSeek | 强性价比、推理/代码能力突出，API 格式兼容 OpenAI/Anthropic 生态         |
