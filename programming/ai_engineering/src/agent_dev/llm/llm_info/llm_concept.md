# LLM 是什么

LLM，全称是 **Large Language Model，大语言模型**。

最简单的理解：

> LLM 是一个基于海量文本训练出来的“预测下一个 token 的模型”。

它不是传统意义上的数据库，也不是搜索引擎，也不是严格的逻辑推理机。

它的核心能力来自：给定前面的文本 -> 预测后面最可能出现的文本

比如：

1. 用户输入：Java Spring Boot 常用于开发……
2. 模型预测：后端服务、微服务、REST API……
3. 用户输入：请帮我写一个 Two Sum 的 Java 解法
4. 模型预测：`public int[] twoSum(...) { ... }`

所以 LLM 的底层目标并不是“理解世界”，而是：根据上下文，生成最合理的后续文本。

但因为训练数据极其庞大，模型参数极多，它在预测文本的过程中学到了很多语言、代码、知识、推理模式和任务模式，于是表现得像是在“理解”。
