# LLM 基本原理与能力边界

这一章用于理解 LLM 的工作方式、适用任务和工程边界。

先了解 [LLM 的基本工作流程](./llm_flow.md)，再理解 [Token 与 Tokenizer](./tokenizer.md)以及 [Token 限制及其工程影响](./token_limit.md)。

能够解释 Token 和上下文窗口，知道 LLM 会产生幻觉且不能直接执行真实动作，并能把 LLM 视为外部服务组件后，即可进入[模型与 Provider 的差异和选型](../llm_compare/llm_compare.md)和[大模型 API 调用](../llm_api/llm_api.md)。无需先掌握 Transformer 数学原理或模型训练细节。
