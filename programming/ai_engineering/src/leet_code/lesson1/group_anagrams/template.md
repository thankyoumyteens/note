# 本课三个核心模板对比

| 题目           | 识别信号                   | HashMap / 数组设计             | 核心模板 |
| -------------- | -------------------------- | ------------------------------ | -------- |
| Two Sum        | 两数之和、target、返回下标 | 数字 -> 下标                   | 补数查找 |
| Valid Anagram  | 两个字符串、字符频率       | 字符 -> 次数                   | 字符计数 |
| Group Anagrams | 字符串数组、分组           | 排序后字符串 -> 原始字符串列表 | 分组 key |

---

## 本课最重要的判断方法

以后看到题目，先问自己这三个问题：

```text
1. 我是不是要快速判断某个东西是否出现过？
   是：HashSet / HashMap

2. 我是不是要记录某个东西出现了几次？
   是：HashMap<Character, Integer> 或 int[26]

3. 我是不是要按照某种规则把对象分组？
   是：Map<Key, List<Value>>
```
