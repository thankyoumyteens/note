# LeetCode / 算法面试

### 第 1 阶段：数组 / 字符串 + HashMap 基础

目标：建立数组遍历、HashMap 查找、字符计数、分组 key 的基础能力。

---

### 第 2 阶段：双指针

目标：掌握左右指针、快慢指针、原地修改。

---

### 第 3 阶段：滑动窗口

目标：掌握连续子数组 / 子字符串问题。

---

### 第 4 阶段：栈 / 队列

目标：掌握 LIFO / FIFO、括号匹配、单调栈、基础队列。

---

### 第 5 阶段：链表

目标：掌握链表指针操作，重点是 dummy、反转、快慢指针。

---

### 第 6 阶段：二分查找

目标：掌握搜索空间、边界收缩、答案二分。

---

### 第 7 阶段：树 DFS / BFS

目标：掌握树的递归、返回值、层序遍历。

---

### 第 8 阶段：堆 / TopK

目标：掌握 PriorityQueue 和 TopK。

---

### 第 9 阶段：图 BFS / DFS

目标：掌握图遍历、网格 DFS/BFS、visited。

---

### 第 10 阶段：回溯

目标：掌握组合、排列、子集、搜索所有解。

---

### 第 11 阶段：动态规划基础

目标：只学面试够用的基础 DP，不钻竞赛 Hard DP。
