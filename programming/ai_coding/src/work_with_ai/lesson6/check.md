# 本课重点检查

你要检查 Claude Code 输出是否符合下面这张表：

| 检查项                         | 要求     |
| ------------------------------ | -------- |
| 是否修改文件                   | 必须没有 |
| 是否有 7 个章节                | 必须有   |
| 是否列出涉及文件               | 必须有   |
| 是否计划修改 `pom.xml`         | 不应该   |
| 是否计划新增依赖               | 不允许   |
| 是否只做 `POST /api/documents` | 必须     |
| 是否返回 `documentId`          | 必须     |
| 是否返回 `title/content`       | 不应该   |
| 是否使用内存存储               | 必须     |
| 是否有测试计划                 | 必须     |
| 是否有回滚方式                 | 必须     |

最重要的是这三点：

```text
1. 不修改 pom.xml
2. 成功响应只返回 documentId
3. 第 6 课不写代码，只给计划
```
