# 本课任务结构

以后你指挥 AI 做开发任务，默认用这个结构：

```text
目标：
背景：
输入：
输出：
限制：
验收标准：
```

这 6 个部分分别解决：

| 部分     | 作用                 |
| -------- | -------------------- |
| 目标     | 告诉 AI 要做什么     |
| 背景     | 告诉 AI 当前上下文   |
| 输入     | 告诉 AI 可以参考什么 |
| 输出     | 告诉 AI 要交付什么   |
| 限制     | 告诉 AI 不要做什么   |
| 验收标准 | 告诉 AI 怎样算完成   |
