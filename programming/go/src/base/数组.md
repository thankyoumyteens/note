# 数组

```go
var arr0 [3]int
// [0 0 0] 0 0 0
fmt.Println(arr0, arr0[0], arr0[1], arr0[2])

// 奇葩问题:
// var arr1 [4]int
// [3]int 和 [4]int是两种不同的数据类型
// cannot use arr1 (variable of type [4]int) as [3]int value in assignment
// arr0 = arr11
```

## 数组初始化

```go
// 初始化每个元素
arr := [3]string{"he", "is", "zhangsan"}
// (he) (is) (zhangsan)
for _, value := range arr {
	fmt.Printf("(%s) ", value)
}

// 初始化指定索引的元素
arr1 := [3]string{1: "is"}
// () (is) ()
for _, value := range arr1 {
	fmt.Printf("(%s) ", value)
}

// 根据初始化的元素个数决定数组的长度
arr2 := [...]string{"he", "is", "zhangsan"}
// (he) (is) (zhangsan)
for _, value := range arr2 {
	fmt.Printf("(%s) ", value)
}
```
