# 变量

```go
package main

import "fmt"

func main() {
    // var 变量名 变量类型 [= 变量值]
	var z int
	var a int = 10
	fmt.Println(z, a)
}
// 输出:
// 0 10
```

## 根据变量值自动推断变量类型

```go
package main

import "fmt"

func main() {
    // var 变量名 = 变量值
	var b = 20
	fmt.Println(b)
}
// 输出:
// 20
```

## 省略 var 关键字

这种声明方式只能在函数体中使用

```go
package main

import "fmt"

func main() {
    // 变量名 := 变量值
	c := 30
	fmt.Println(c)
}
// 输出:
// 30
```

## 多变量声明

```go
package main

import "fmt"

func main() {
	// 多变量声明
	var d, e, f int
	var g, h, i = 1, 2, 3
	j, k, l := 4, 5, 6
	fmt.Println(d, e, f, g, h, i, j, k, l)
}
// 输出:
// 0 0 0 1 2 3 4 5 6
```
