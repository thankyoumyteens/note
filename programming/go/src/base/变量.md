# 变量

定义变量的 3 种方式:

- 方式 1: var 变量名 变量类型 \[= 变量值\]
- 方式 2: var 变量名 = 变量值
- 方式 3: 变量 := 变量值, 这种方式不能用于全局变量

```go
// 方式1
var a int
a = 1
var b int = 1
// 定义了局部变量, 不使用会报错
fmt.Println(a, b)

// 方式2
var c = 1
fmt.Println(c)

// 方式3
d := 1
fmt.Println(d)
```

## 定义多个变量

```go
var a, b, c = "a", true, 100
fmt.Println(a, b, c)
var (
    d = 1
    e = 2
)
fmt.Println(d, e)
```

## 匿名变量

go 中, 定义了变量就必须要使用, 但是也有不需要使用变量的情况, 比如函数的返回值, 这种情况可以使用匿名变量。

匿名变量使用下划线定义:

```go
func getTwoValue() (int, bool) {
	return 1, true
}

func main() {
    // 不使用bool类型的返回值
	value, _ := getTwoValue()
	fmt.Println(value)
}
```
