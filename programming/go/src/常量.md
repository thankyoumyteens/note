# 常量

```go
package main

import "fmt"

func main() {
	// const 常量名 常量类型 = 常量值
	const a int = 10
	// const 常量名 = 常量值
	const b = 10
	fmt.Println(a, b)
}
// 输出:
// 10 10
```

## 常量作为枚举

```go
package main

import "fmt"

func main() {
	const (
		Teacher = 0
		Student = 1
	)
	fmt.Println(Teacher, Student)
}
// 输出:
// 0 1
```

## iota

iota 在 const 关键字内出现时将被重置为 0, const 中每新增一行常量声明将使 iota 加 1

```go
package main

import "fmt"

func main() {
	const (
		a = iota
		b
		c
		d
		e
	)
	const (
		A = iota
		B
		C
		D
	)
	fmt.Println(a, b, c, d, e)
	fmt.Println(A, B, C)
}
// 输出:
// 0 1 2 3 4
// 0 1 2
```
