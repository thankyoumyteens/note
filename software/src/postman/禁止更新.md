# 禁止更新

## 安装 asar

```sh
npm install -g asar
```

## 解压源码

打开安装目录下 resources 目录, 解压 .asar 文件 ：

```sh
asar extract app.asar ./app
```

## 修改

修改 app/services/AutoUpdaterService.js 文件:

```js
// 替换isAppUpdateEnabled:
isAppUpdateEnabled = function () {
  return false;
};
```

修改 app/js/scratchpad/scratchpad.js 文件:

```js
// 找到
(this.state = { showBanner: !0 }),
// 替换成
(this.state = { showBanner: !1 }),
```

## 重新打包

```sh
asar pack app app.asar
```

删除 app 文件夹
