# 替换

1. re.sub("abcd","m",string) 将abcd全部替换为m
2. re.sub("abcd","m",string,2) 只替换前两个字符
3. re.sub("\d",func,string) 用函数的返回值替换

```python
# 将字符串中的数字替换成%
import re

def parse(matched):
    s = matched.group('v1')
    end = matched.group('v2')
    v = matched.group('v')
    return s + '%' + end

s = 'A23GG4HFD567GGG4846'
print(re.sub('(?P<v1>[A-Z])(?P<v>\d+?)(?P<v2>[A-Z])', parse, s))
```
