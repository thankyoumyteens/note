# 垂直翻转

```
ffmpeg -i in.mp4 -vf vflip -y out.mp4
```

# 水平翻转

```
ffmpeg -i in.mp4 -vf hflip -y out.mp4
```

# 顺时针旋转90度

```
ffmpeg -i in.mp4 -vf transpose=1 out.mp4
```

# 顺时针旋转180度

```
ffmpeg -i in.mp4 -vf "transpose=1,transpose=1" out.mp4
```

# 逆时针旋转90度

```
ffmpeg -i in.mp4 -vf transpose=2 out.mp4
```
