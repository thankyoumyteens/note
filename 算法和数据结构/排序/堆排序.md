# 堆排序

最小二叉堆中, 根节点最小, 将数组中的元素放入堆中再依次取出根节点就是从小到大的顺序

```java
public void heapSort(Comparable[] array) {
      Heap heap = new Heap(array.length);
      for (Comparable item : array) {
          heap.insert(item);
      }
      for (int i = 0; i < array.length; i++) {
          array[i] = heap.extractMin();
      }
  }
```
