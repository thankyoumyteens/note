# 二分查找法

假设数组 升序排列，查找目标值 `target`：

1. 初始化左右指针 `left`（指向数组起始索引 0）和 `right`（指向数组末尾索引 n-1）
2. 计算中间索引 `mid = left + (right - left) / 2`（避免 left + right 溢出）
3. 比较 `nums[mid]` 与 `target`：
   - 若 `nums[mid] == target`：找到目标，返回 `mid`；
   - 若 `nums[mid] < target`：目标在右半区间，更新 `left = mid + 1`
   - 若 `nums[mid] > target`：目标在左半区间，更新 `right = mid - 1`
4. 重复步骤 2-3，直到 `left > right`（查找区间为空，目标不存在），返回 `-1`

## 适用条件

1. 数组必须有序（升序/降序，若无序需先排序，排序时间 O(nlogn)
2. 数组需支持随机访问, 链表不适用，因为链表访问中间元素需 O(n) 时间
3. 无重复元素(或允许重复但只需返回任意匹配索引，若需返回首个/最后一个匹配索引，需微调算法)
