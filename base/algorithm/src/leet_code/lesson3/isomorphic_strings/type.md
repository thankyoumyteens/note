# 题型识别信号

看到这些关键词：

```text
isomorphic
同构字符串
字符替换
每个字符对应另一个字符
映射关系不能冲突
```

优先想到：

```text
双向映射
```

这题不是计数题。它不关心某个字符出现多少次，而关心：

```text
s 中的某个字符，是否始终映射到 t 中同一个字符
t 中的某个字符，是否也只被 s 中一个字符映射
```

为什么需要双向映射？

因为只做单向映射会漏掉这种情况：

```text
s = "ab"
t = "aa"
```

如果只检查：

```text
a -> a
b -> a
```

单向看似没冲突，但实际上 `a` 和 `b` 都映射到 `a`，不是一一对应，所以应该返回 `false`。
