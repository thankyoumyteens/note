# 数学归纳法

<!--
\begin{align}
& 数学归纳法是一种用于证明与自然数 n 有关的命题 P(n) 的正确性的重要方法。\\
\\
& 运用步骤 \\
& 1、明确命题：首先要清晰地表述出需要证明的与自然数 n 有关的命题 P(n) \\
& 2、完成基础步骤：验证当 n = n_0 时，命题 P(n_0) 是否成立 \\
& 3、完成归纳步骤：\\
& \quad 1、作出归纳假设，即假设当 n = k（k\geq n_0）时命题 P(k) 成立 \\
& \quad 2、以归纳假设为条件，通过合理的推导和变换，证明当 n = k + 1 时命题 P(k + 1) 也成立 \\
& 4、得出结论：由于基础步骤证明了 n = n_0 时命题成立，\\
& \;\;\;\;\, 归纳步骤又证明了如果 n = k 时命题成立，那么 n = k + 1 时命题也成立，\\
& \;\;\;\;\, 所以可以得出命题 P(n) 对于所有大于等于 n_0 的自然数 n 都成立 \\
\\
& 示例 \\
& 证明 1 + 2 + 3 + \cdots + n = \frac{n(n + 1)}{2} 对所有正整数 n 成立 \\
& {\color{Green} // 1、明确命题} \\
& 设 P(n) 为命题1 + 2 + 3 + \cdots + n = \frac{n(n + 1)}{2}，其中 n 为正整数 \\

\end{align}

<!--
示例：证明 1 + 2 + 3 + \cdots + n = \frac{n(n + 1)}{2} 对所有正整数 n 成立
1. **明确命题**：
设 P(n) 为命题“1 + 2 + 3 + \cdots + n = \frac{n(n + 1)}{2}”，其中 n 为正整数。
2. **基础步骤**：
当 n = 1 时，左边 = 1，右边 =\frac{1\times(1 + 1)}{2} = 1，左边等于右边，所以 P(1) 成立。
3. **归纳步骤**：
- **归纳假设**：假设当 n = k（k\geq 1，k 为正整数）时，命题 P(k) 成立，即 1 + 2 + 3 + \cdots + k = \frac{k(k + 1)}{2}。
- **证明 P(k + 1) 成立**：
当 n = k + 1 时，左边 = 1 + 2 + 3 + \cdots + k + (k + 1)。
由归纳假设可知 1 + 2 + 3 + \cdots + k = \frac{k(k + 1)}{2}，所以左边 =\frac{k(k + 1)}{2} + (k + 1)。
对其进行化简：
\[
\begin{align*}
\frac{k(k + 1)}{2} + (k + 1)&=\frac{k(k + 1)}{2} + \frac{2(k + 1)}{2}\\
&=\frac{k(k + 1)+2(k + 1)}{2}\\
&=\frac{(k + 1)(k + 2)}{2}\\
&=\frac{(k + 1)[(k + 1) + 1]}{2}
\end{align*}
\]
这与右边（当 n = k + 1 时，右边为 \frac{(k + 1)[(k + 1) + 1]}{2}）相等，所以当 n = k + 1 时，命题 P(k + 1) 成立。
4. **得出结论**：
由基础步骤可知 P(1) 成立，再由归纳步骤可知，如果 P(k) 成立，那么 P(k + 1) 也成立。所以，根据数学归纳法，命题 P(n) 对于所有正整数 n 都成立。
