# CISC 和 RISC

|                    | CISC                           | RISC                         |
| ------------------ | ------------------------------ | ---------------------------- |
| 指令系统设计       | 复杂                           | 精简                         |
| 指令数量           | 一般大于 200 条                | 一般小于 100 条              |
| 指令字长           | 长度不固定                     | 长度固定                     |
| 可访存指令         | 无限制, 多种指令都可以访存     | 只有 LOAD 和 STORE 访存      |
| 不同指令的执行时间 | 执行时间相差很大               | 一般在一个机器周期内完成     |
| 不同指令的使用频率 | 使用频率相差很大               | 每条指令的使用频率接近且较高 |
| 通用寄存器的数量   | 少                             | 多                           |
| 目标代码           | 难以优化编译成效率高的目标代码 | 容易编译成效率高的目标代码   |
| 控制方式           | 多数为微程序控制               | 多数为组合逻辑控制           |
| 指令流水线         | 有, 但无强制要求               | 强制必须有                   |

时钟周期: 一个时钟脉冲所需要的时间, 是 CPU 和其它单片机的基本时间单位。

机器周期: 通常用从内存中读取一个指令字的最短时间来规定机器周期, 即 CPU 完成一个基本操作所需的时间。
