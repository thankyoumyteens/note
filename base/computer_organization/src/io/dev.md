# 外部设备

## 输入设备

键盘、鼠标、触屏、麦克风、扫描仪、摄像头等。

## 输出设备

### 显示器

- 屏幕大小: 以对角线长度来表示
- 灰度级: 就是指纯白、纯黑以及两者中的一系列从黑到白的过渡色的多少
- 分辨率: 即像素个数，以纵横方向像素个数乘积表示，如: 800 x 600
- 刷新频率: 光点只能维持很短的时间，为此必须在光点消失之前再重新显示一遍，这个过程称之为刷新，单位时间内整个屏幕中所有光点的刷新次数称之为刷新频率

### 打印机

- 喷墨打印机
- 针式打印机
- 激光打印机

### 外存储器

- 硬磁盘存储器
- 光盘

光盘的特点: 存储密度高，成本低、容量大、携带方便、存储期长等。

光盘的分类:

- CD-ROM: 只读光盘
- CD-R: 只可写入一次的光盘
- CD-RW: 可读可写光盘，可以重复读写
- DVD-ROM: 高容量的 CD-ROM
