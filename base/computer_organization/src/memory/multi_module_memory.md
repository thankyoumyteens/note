# 多模块存储器

多模块存储器是一种旨在提高计算机系统中数据访问速度的技术。多模块存储器主要分为两大类：单体多字存储器和多体并行存储器。

## 单体多字存储器

单体多字存储器是指存储器中只有一个存储体，但是每个存储单元可以存储多个字。

特点: 在一个存取周期内可以访问多个字, 但如果遇到转移指令, 或者操作数不连续存放的情况则效率提升不明显。

## 多体并行存储器

多体并行存储器由多个具有相同容量和存取速度的存储模块组成，每个模块都有自己独立的读写控制电路、地址寄存器和数据寄存器。

根据数据组织的方式，多体并行存储器又可以细分为高位交叉编址和低位交叉编址两种形式。

- 高位交叉编址：在这种方式中，地址的高位部分用于确定数据所在的存储模块，低位部分用于指定模块内部的具体位置。这种方式适用于数据访问模式较为固定的情况
- 低位交叉编址：使用地址的低位来决定数据所在的模块，高位则指示模块内的具体位置。这种方式更适合于连续地址的快速访问，因为它可以在不同的存储模块之间交替存取数据，从而有效减少等待时间
