# 计算机的性能指标

## 存储容量

存储体由多个存储单元组成, 每个存储单元又包含多个存储元件, 每个存储元件能保存一位二进制数。所以一个存储单元能保存一串二进制数, 这串二进制数称为一个存储字, 这串二进制数的长度称为存储字长。

存储容量 = 存储单元的个数 × 存储字长, 单位: 位(bit)

MAR 的位数反映了最大存储单元的个数(一个存储单元对应一个内存地址, 如果 MAR 有 4 位, 则存储单元最多有 2<sup>4</sup>=16 个)。

MDR 的位数等于存储字长。

## 机器字长

机器字长: 进行一次整数运算所能处理的最大二进制位数。一般是一个字节的整数倍。它等于 CPU 内部的运算器位数和通用寄存器的宽度。

## 运算速度

吞吐量: 单位时间内的数据传输量。

响应时间: 从用户向计算机发出一个请求开始, 到计算机返回这个请求对应的结果为止的时间。

时钟周期: 两个时钟脉冲之间的时间间隔, 时钟周期是机器中的最小时间单位。时钟脉冲是一个周期性的电信号, 它在高电平和低电平之间交替变化, 每个周期的转换点用来触发 CPU 内部的各种操作。

主频: 每秒钟有多少个时钟周期, 主频 = 时钟周期的倒数。

CPI: 执行一条指令所需要的时钟周期数。

某一个指令序列的 CPI: 这个序列执行完所有指令所需要的时钟周期数除以这个序列的指令总数。

MIPS: 每秒执行多少百万条指令。

FLOPS: 每秒的浮点运算次数。

## 例题 1

机器 A 的主频为 800MHz,某程序在 A 上运行需要 12s。现在硬件设计人员想设计机器 B，希望该程序在 B 上的运行时间能缩短为 8s,使用新技术后可使 B 的主频大幅度提高， 但在 B 上运行该程序所需的时钟周期数为在 A 上的 1.5 倍。则机器 B 的主频至少应为？

解:

该程序在机器 A 上需要的时钟周期数为 12×800M=9600M，因为在机器 B 上运行该程序所需的时钟周期数为在 A 上的 1.5 倍，所以在 B 上需要的时钟周期数为 9600M×1.5=14400M=14.4G，要求运行时间为 8s，故 B 的时钟频率为 14.4G÷8=1.8GHz。
