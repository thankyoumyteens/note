# 计算机的性能指标

## 存储容量

存储容量 = 存储单元的个数 × 存储字长, 单位: 位(bit)

- MAR 的位数反映了最大存储单元的个数(因为 1 个存储单元对应 1 个内存地址)
- MDR 的位数等于存储字长

## 机器字长

机器字长: 进行一次整数运算所能处理的最大二进制位数。一般是一个字节的整数倍。

## 运算速度

吞吐量: 单位时间内的数据传输量。

响应时间: 从用户向计算机发出一个请求开始, 到计算机返回这个请求对应的结果为止的时间。

时钟周期: 两个时钟脉冲之间的时间间隔, 时钟周期是机器中的最小时间单位。时钟脉冲是一个周期性的电信号，它在高电平和低电平之间交替变化，每个周期的转换点用来触发 CPU 内部的各种操作。

主频: 每秒钟有多少个时钟周期, 主频 = 时钟周期的倒数。

CPI: 执行一条指令所需要的时钟周期数。

某一个指令序列的 CPI: 这个序列执行完所有指令所需要的时钟周期数除以这个序列的指令总数。

MIPS: 每秒执行多少百万条指令。

FLOPS: 每秒的浮点运算次数。
