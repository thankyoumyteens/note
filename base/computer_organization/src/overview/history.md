# 计算机发展史

| 发展阶段 | 时间        | 技术               | 速度(次/秒) |
| -------- | ----------- | ------------------ | ----------- |
| 1        | 1946 ~ 1957 | 电子管             | 40,000      |
| 2        | 1958 ~ 1964 | 晶体管             | 200,000     |
| 3        | 1965 ~ 1971 | 中、小规模集成电路 | 1,000,000   |
| 4        | 1972 ~ 1977 | 大规模集成电路     | 10,000,000  |
| 5        | 1978 ~ 至今 | 超大规模集成电路   | 100,000,000 |

微处理器的出现和发展: 把运算器和控制器合并到一起就构成了微处理器。微处理器经历了 4 位、8 位、16 位、32 位、和 64 位微处理器的几个发展阶段(xx 位处理器: 处理器一次最多能处理的二进制位数)。
