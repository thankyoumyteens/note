# 习题

1. 假设基准程序 A 在某计算机上的运行时间为 100s, 其中 90s 为 CPU 时间, 其余是 I/O 时间。若 CPU 速度提高 50%, I/O 速度不变, 则运行基准程序 A 所耗费的时间是:
    ```
    CPU时间: 90 / (1 + 50%) = 60s
    I/O时间不变
    总时间： 60 + 10 = 70s
    ```
2. 程序P在机器M上的执行时间是20s, 编译优化后, P执行的指令数减少到原来的70%, 而CPI增加到原来的1.2倍, 则P在M上的执行时间是:
    ```
    设: M的原时钟周期为a, 原CPI为b, P的原指令条数为c
    则P的原执行时间为: a × b × c = 20s

    编译优化后, P的执行时间为: 
      a × (1.2 × b) × (0.7 × c) 
    = (1.2 × 0.7) × (a × b × c) 
    = 1.2 × 0.7 × 20 = 16.8s
    ```
