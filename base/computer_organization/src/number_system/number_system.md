# 数据的表示和运算

每个数位所用到的不同符号的个数称为基数。比如二进制, 只有两个不同的符号 0 和 1。

## B 进制的表示方法

整数部分:

A<sub>n</sub> × B<sup>n</sup> + ... + A<sub>2</sub> × B<sup>2</sup> + A<sub>1</sub> × B<sup>1</sup> + A<sub>0</sub> × B<sup>0</sup>

小数部分:

A<sub>-1</sub> × B<sup>-1</sup> + A<sub>-2</sub> × B<sup>-2</sup> + A<sub>-3</sub> × B<sup>-3</sup> + ... + A<sub>-n</sub> × B<sup>-n</sup>

比如十进制数 12.34 可以表示成:

1 × 10<sup>1</sup> + 2 × 10<sup>0</sup> + 3 × 10<sup>-1</sup> + 4 × 10<sup>-2</sup>
