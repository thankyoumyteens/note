# 总线的性能指标

- 总线周期: 一次总线操作所需要的时间
- 总线工作频率: 总线周期的倒数
- 总线时钟周期: 就是机器时钟周期
- 总线时钟频率: 总线时钟周期的倒数
- 总线宽度: 总线上能够同时传输的数据位数
- 总线带宽: 单位时间内总线上可以传输的数据位数
- 总线复用: 一条信号线上分时传输两种信号(数据信号, 地址信号)
- 信号线数: 地址总线, 数据总线和控制总线的总数

假设某系统总线在一个总线周期中并行传输 4 个字节信息，一个总线周期占用两个时钟周期，总线时钟频率为 10MHz,则总线带宽是多少？

时钟周期为:1/10MHz=0.1 微秒

1 个总线周期包含 2 个时钟周期，传输 4 字节信息, 故总线带宽为: 4B/(2 \* 0.1) = 4B / (0.2x10^-6) = 20MB/s
