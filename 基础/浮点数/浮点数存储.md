# 科学计数法

N = M × R^e

- R称为基数
- e称为指数
- M称为尾数

# 浮点数的组成

一个浮点数由阶码, 尾数和符号位组成

```
[符号S][阶码E][尾数M]
```

- 尾数: 使用定点小数表示, 尾数的位数决定了浮点数的精度
- 阶码: 用来表示指数, 使用整数移码形式, 决定了浮点数的表示范围
- 基数: 固定为2

# IEEE754标准

- 32位单精度浮点数: 尾数23位(0-22), 阶码8位(23-30), 符号位1位(31)
- 64位双精度浮点数: 尾数52位(0-51), 阶码11位(52-62), 符号位1位(63)
- 尾数的整数位默认为1

# 阶码和指数的关系

- 单精度浮点数: 阶码 = 指数 + 127
- 双精度浮点数: 阶码 = 指数 + 1023

# 求浮点数的真值

- 单精度浮点数: x = (-1)^S × (1.M) × 2^{E-127}
- 双精度浮点数: x = (-1)^S × (1.M) × 2^{E-1023}
