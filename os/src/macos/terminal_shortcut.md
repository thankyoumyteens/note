# 终端中的快捷键

| 操作                 | 快捷键            |
| -------------------- | ----------------- |
| 复制                 | `Command + C`     |
| 粘贴                 | `Command + V`     |
| 光标移动到行首       | `Control + A`     |
| 光标移动到行尾       | `Control + E`     |
| 光标向前移动一个单词 | `Option + 左箭头` |
| 光标向后移动一个单词 | `Option + 右箭头` |
| 光标向后移动一个单词 | `Option + 右箭头` |
