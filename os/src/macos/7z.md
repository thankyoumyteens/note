# 7-zip

### 1. 下载

[7z2301-mac.tar.xz](https://www.7-zip.org/a/7z2301-mac.tar.xz)

### 2. 解压:

```sh
mkdir 7z2301
tar -xvf 7z2301-mac.tar.xz -C 7z2301/
```

### 3. 配置环境变量:

```sh
# 打开配置文件
vim ~/.zshrc

# 在最后一行添加
export PATH=/Users/walter/walter/software/7z2301:$PATH

# 使环境变量生效
source ~/.zshrc
```

### 4. 验证

```sh
7zz -h
```
