# 安装常用软件

```sh
# chrome
brew install --cask google-chrome
# ungoogled-chromium
brew install --cask eloston-chromium
# subversion
brew install subversion
# vlc
brew install vlc
# 鼠标滚轮
brew install mos
# vscodium
brew install --cask vscodium
# mdbook
brew install mdbook
# ffmpeg
brew install ffmpeg
# wget
brew install wget
```
