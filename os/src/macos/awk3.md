# 模式和动作

模式(pattern)可以是：

- 正则表达式（如 /^[0-9]+$/，匹配纯数字行）；
- 关系表达式（如 $2 > 30，第二个字段大于 30）；
- 行号范围（如 NR == 5 匹配第 5 行，NR > 10 && NR < 20 匹配 11-19 行）；
- 空模式（不写模式，默认匹配所有行）。

动作(action)是用 `{}` 包裹的语句块，可包含打印（print/printf）、变量运算、条件判断（if-else）、循环（for/while）等逻辑。若动作省略（只写模式），默认动作是 `print $0`（打印整行）。

示例：

```sh
# 若第二个字段>18，打印姓名
awk '$2 > 18 { print $1 "是成年人" }' demo.txt
# 匹配含"张三"的行，计数器+1
awk '/张三/ { count++ } END {print "总共有" ,count, "个张三";}' demo.txt
```
