# RTX5090Laptop玩游戏频繁崩溃

## 原因

瞬时高频导致的掉电：你的移动端 5090 在渲染高画质特效时，核心频率瞬间冲刺，导致电压跟不上，GPU 瞬间宕机。

## 解决方法

安装 MSI Afterburner 并配置：

1. 打开曲线编辑器（Curve Editor）。
2. 在横轴找到 **`900mV`**，鼠标**左键点击**那条线上的小方块节点（选中后方块会变大一点）。
3. **按下键盘上的 `L` 键。**
4. 此时，你会看到在这个节点上出现了一条**黄色的垂直虚线**。这意味着显卡被彻底“焊死”在这个电压和频率上了，无论游戏怎么刺激，它都不准越过这条线。
5. 关掉曲线窗口，回到主界面点击右下角的 **Apply**。

_注：测试完后，如果你想解除锁定，再次选中那个点按 `L` 键即可取消黄线。_

保存并自启该配置：

1. 在 MSI Afterburner 主界面下方，点击 “保存 (Save)”，然后点击右侧的数字 1。接着点击左侧边栏的 “Windows 图标”（Startup）。这样每次开机，这个“稳如老狗”的频率曲线就会自动加载，不用每次进游戏前都手动调了。

## 为了让它默默在后台守护你的 5090，同时不碍眼，你需要做以下设置

- 日常隐藏： 不要点右上角的 X（虽然默认点 X 也是最小化，但有些版本会直接关闭）。养成点 最小化 的习惯，让它缩进右下角的系统托盘里。

设置开机自启且静默运行：

1. 点击主界面左侧的 齿轮图标 (Settings)。
2. 在 常规 (General) 选项卡里，勾选 随 Windows 启动 (Start with Windows)。
3. 同样勾选 启动后最小化 (Start minimized)。
4. 点击确定。

这样设置完后，你这台电脑每次开机，Afterburner 就会悄无声息地在后台把你的 5090 安排得明明白白，你甚至感觉不到它的存在，但你再也不会遇到“白屏叮当”的崩溃了。它占用的 CPU 和内存资源极小，对你的系统性能毫无影响。
