# expr

expr 是一款表达式计算工具, 使用它能完成表达式的求值操作。

```sh
#!/bin/sh

i=10
echo $i
i=`expr $i + 10`
echo $i #20
i=`expr $i + 100`
echo $i #120
```

- 表达式和运算符之间要有空格
- 完整的表达式要被 ` 包含

# let

let命令是bash内置命令, 可以实现简单的算术以及逻辑运算

```sh
#!/bin/sh

i=10
let i=i+10
echo $i  #20
let "i=i+100"
echo $i #120
```

# []

`[]`相当于let命令

```sh
#!/bin/sh

i=10
i=$[i+10]
echo $i #20
i=$[i+100]
echo $i #120
```

# (())

`(())`等价于`[]`
